1. install the dll in your BepInEx plugins folder

2. the mod should've made a config folder

3. close the game, and go into the config file (com.doug.texturereplacer.cfg)

4. get your image (preferably a .png) and open the cfg file

5. on the part where it says "Texture path =", paste your file path

(e.g. Texture path = C:\Users\you\Downloads\poopootest.png)

6. play and then regret that you've installed this mod
